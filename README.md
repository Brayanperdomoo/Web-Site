# 📘 Blog Educativo de Etiquetas HTML

## 📖 Descripción

El **Blog Educativo de Etiquetas HTML** es una página web diseñada con el objetivo de enseñar y explicar las etiquetas más importantes del lenguaje **HTML** de forma clara, organizada y visualmente atractiva.

El proyecto presenta las etiquetas agrupadas por **categorías**, permitiendo a los usuarios navegar fácilmente entre ellas mediante una **barra de navegación interactiva**. Cada etiqueta incluye una breve explicación que facilita la comprensión de su uso dentro del desarrollo web.

Este blog busca servir como un **recurso educativo introductorio** para estudiantes que están iniciando en el mundo del **desarrollo web**.

---

## 🎯 Objetivos del Proyecto

- Explicar de forma sencilla y visual las principales **etiquetas HTML**.
- Organizar el contenido por **categorías** para mejorar la comprensión.
- Aplicar **buenas prácticas de desarrollo web**.
- Implementar una interfaz **moderna, limpia y fácil de navegar**.
- Utilizar tecnologías web actuales como **Bootstrap, HTML5, CSS3 y JavaScript**.

---

## 🧱 Tecnologías Utilizadas

El proyecto fue desarrollado utilizando las siguientes tecnologías:

- **HTML5** → Estructura del contenido.
- **CSS3** → Estilos y diseño visual.
- **Bootstrap 5** → Framework para diseño responsivo.
- **JavaScript** → Interactividad y navegación dinámica.

---

## 🎨 Características del Blog

- Diseño moderno con **paleta de colores azules**.
- Interfaz **limpia y agradable a la vista**.
- Navegación por **categorías de etiquetas HTML**.
- Desplazamiento suave entre secciones.
- Organización clara del contenido.
- Uso de **tarjetas (cards)** para explicar cada etiqueta.

---

### 📁 Descripción de Carpetas

- **assets/images** → Contiene imágenes utilizadas en el blog.
- **css/styles.css** → Archivo donde se definen los estilos del sitio.
- **js/script.js** → Archivo que contiene las funciones de JavaScript.
- **index.html** → Archivo principal que estructura la página web.

---

## 📚 Categorías de Etiquetas HTML

Las etiquetas del blog están organizadas en las siguientes categorías:

- **Estructura**
- **Texto**
- **Listas**
- **Multimedia**
- **Formularios**

Cada categoría contiene etiquetas explicadas de manera clara para facilitar el aprendizaje de HTML.

---

El proyecto puede abrirse en cualquier navegador web moderno.
